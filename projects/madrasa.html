Madrasa Management Web Application

Project Overview

A web-based management system developed for managing madrasa operations including student admissions, fees, batches, notices, results and administration.

Features

  • Student Admission Management
  • Student Records
  • Batch Management
  • Fee Management
  • Result Management
  • Notice Board
  • Admin Dashboard

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • Node.js
  • Express.js
  • MySQL
  • Prisma ORM

Challenges Faced

Designing the database structure, implementing relationships between students, batches and fees, and creating an admin approval workflow.

What I Learned

Full-stack development, database design, backend APIs, authentication concepts, and project management.

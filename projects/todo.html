Todo Manager App

Project Overview

Todo Manager App is a task management application that allows users to create, update, delete and organize daily tasks efficiently.

Features

  • Add New Tasks
  • Edit Existing Tasks
  • Delete Tasks
  • Mark Tasks as Complete
  • Simple User Interface

Technologies Used

  • HTML
  • CSS
  • JavaScript

Challenges Faced

Managing task states and ensuring smooth user interaction while keeping the interface simple.

What I Learned

DOM Manipulation, Event Handling and Frontend Development fundamentals.

Influencer Fraud Detection

Project Overview

A machine learning-based system designed to identify fake influencers by analyzing engagement patterns, follower growth, and audience interaction metrics.

Features

  • Engagement Analysis
  • Anomaly Detection
  • Data Visualization
  • Machine Learning Predictions

Technologies Used

  • Python
  • Pandas
  • NumPy
  • Scikit-Learn
  • Machine Learning
  • Data Analysis

Challenges Faced

Handling large datasets, selecting suitable machine learning algorithms, and interpreting engagement metrics accurately.

What I Learned

Data preprocessing, anomaly detection, machine learning workflows, and model evaluation.
